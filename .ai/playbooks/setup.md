# Setup Playbook

Use this playbook when onboarding or reshaping the assistant system.

## Goal

Turn this starter into a personalized executive-assistant vault without overbuilding it.

## Order

1. Read the runtime docs plus this playbook.
2. Run the onboarding behavior from `.ai/skills/assistant-onboarding/SKILL.md`.
3. Update `.ai/memory/` first.
4. Create only the folders and guides the user actually needs.
5. Add runtime-specific behavior only after the shared core is correct.

## Success Test

The vault should feel more personal and more useful after onboarding, but not heavier.
